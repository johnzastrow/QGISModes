description=Switchable simplified QGIS interfaces ("modes") for task-focused work.
about=QGIS Modes lets you define several simplified QGIS interfaces — "modes"
    such as Data Editing, Analysis, Raster Processing, or Output Creation — and
    switch between them at runtime. It builds on QGIS Light (by Serkan Girgin),
    extending its single config-driven simplified interface into multiple named,
    selectable configurations, with a planned visual tool for authoring them.
    Requires the QuickMapServices and DataPlotly plugins (auto-installed via
    the QGIS plugin-dependency prompt). This release is an early skeleton; see
    the project documentation.
